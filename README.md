# Animated Tab Navigation Bar - CodingLab

Um componente de barra de navegação inferior (Tab Navigation) moderno, minimalista e altamente interativo desenvolvido com HTML5 e CSS3. O projeto destaca a aplicação de microinterações fluidas e uma transição de destaque deslizante (sliding indicator) que acompanha a seleção do usuário entre as abas.

## Tecnologias Utilizadas

* HTML5: Estruturação semântica do menu de navegação utilizando listas e contêineres de links.
* CSS3: Estilização completa, uso de cantos arredondados, gerenciamento de cores e sombras projetadas.
* Animações em CSS: Aplicação de transições dinâmicas (`transition`) e propriedades de transformação (`transform`) para criar o efeito de deslizamento e alteração de estado dos botões.

## Funcionalidades e Técnicas Aplicadas

* Indicador Deslizante Animado (Active Indicator): Um bloco de destaque com fundo azul suave e cantos arredondados que desliza horizontalmente de forma fluida para se posicionar sob a aba ativa.
* Efeito de Estado Ativo Dinâmico: 
  * **Aba Selecionada ("Home"):** O texto ganha a cor azul institucional da marca e uma tipografia mais marcante.
  * **Abas Inativas ("User", "Setting", "Message"):** Mantêm uma estética minimalista em tons de cinza escuro para direcionar o foco do utilizador.
* Design Flutuante Elegante: O contêiner principal branco possui cantos fortemente arredondados (`border-radius`) e uma sombra projetada (`box-shadow`) sutil que confere profundidade sobre o plano de fundo azul vivo.
* Alinhamento e Simetria: Uso de CSS Flexbox para distribuir os itens de forma perfeitamente equilibrada ao longo da barra.
